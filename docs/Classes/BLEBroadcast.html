@method setMeshCast:data:

Set advertising data. Supported advertising data types are CBUUIDsKey. When in the foreground, an application can utilize up to 16 bytes data per time, While an application is in the background, applications that have specified the “bluetooth-peripheral” background mode will only advertise uuid while in the background.
